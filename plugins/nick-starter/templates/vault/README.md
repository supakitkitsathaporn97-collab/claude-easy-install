# Your second brain

This folder is a ready-to-use note vault, created by your assistant.

**To browse it visually:** open the free [Obsidian](https://obsidian.md) app →
**Open folder as vault** → pick this folder. (If Obsidian was installed for
you already, just launch it and open this folder once.)

You don't have to use Obsidian at all — these are plain text files, and your
assistant reads and writes them either way.

| Folder | What goes there |
|---|---|
| `00-Inbox` | Quick captures — sort later |
| `01-Daily` | One note per day (your assistant's `daily-note`) |
| `02-Notes` | Evergreen notes and ideas |
| `03-Projects` | One note per project or goal |
| `04-People` | People, clients, contacts |
| `05-Resources` | Links, references, attachments |
| `99-Archive` | Finished or retired notes |

Start at [Home](Home.md).
