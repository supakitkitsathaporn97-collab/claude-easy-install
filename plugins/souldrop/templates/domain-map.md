# Domain map — profession/goal → candidate skills

Used by `forge-skills` as the STARTING point for deriving custom skills.
The user's free-text goal always beats this map: match the row(s), then keep
only candidates that serve their actual goal, and invent better ones when the
goal calls for it. Pick 3–5 total, tightly scoped, kebab-case names.

Skip anything the core skills already cover (remember, recall, daily-note,
work-smart, learn-from-mistakes, personal, leader).

| Profession / domain | Typical goals | Candidate skills (name — one-line job) |
|---|---|---|
| Photographer / videographer | book more clients, deliver faster | `client-inquiry-reply` — draft warm replies to booking inquiries · `quote-builder` — draft a priced quote from job details · `shoot-shotlist` — plan shots from venue/brief · `delivery-email` — hand-off message when photos/video are ready |
| Content creator / social media | grow audience, post consistently | `weekly-content-plan` — 7-day post plan from one topic · `caption-writer` — platform-fit captions + hashtags · `video-hook-writer` — 5 hook options for a short video · `comment-reply-helper` — on-brand replies to comments/DMs |
| Small business / shop owner | more sales, less admin | `customer-reply` — answer product/price/shipping messages · `promo-writer` — announcement for a sale or new product · `simple-invoice` — clean invoice text from order details · `review-response` — gracious replies to good AND bad reviews |
| Freelancer (any craft) | win projects, get paid on time | `proposal-writer` — short winning proposal from a job post · `rate-negotiator` — polite scripts for scope/rate pushback · `payment-reminder` — firm-but-friendly overdue notes · `project-status-update` — weekly client update from bullet notes |
| Office / admin / manager | clear inbox, run meetings, report up | `email-drafter` — professional emails from one-line intent · `meeting-summary` — notes → decisions + action items · `weekly-report` — status report from scattered notes · `task-prioritizer` — today's list from a brain-dump |
| Sales / marketing | more leads, better follow-up | `outreach-message` — personalized first-touch messages · `follow-up-sequence` — 3-step follow-up cadence · `objection-answers` — responses to common objections · `case-study-draft` — customer win → short case study |
| Teacher / trainer / coach | prep faster, engage students | `lesson-planner` — lesson plan from topic + level + time · `quiz-maker` — questions + answer key from material · `parent-update` — progress notes home, kind and clear · `feedback-writer` — constructive student feedback |
| Student / researcher | learn faster, write better | `study-summarizer` — dense material → study notes · `essay-outliner` — thesis + structure from a prompt · `flashcard-maker` — Q/A cards from any text · `citation-helper` — tidy references in the required style |
| Real estate / broker | more listings, faster closings | `listing-writer` — compelling listing from property facts · `viewing-follow-up` — after-viewing messages that close · `market-update-post` — local market post for socials |
| Food / restaurant / café | fill seats, run socials | `menu-describer` — appetizing dish descriptions · `daily-special-post` — quick social post for today's special · `reservation-reply` — booking confirmations & changes |
| Developer / tech | ship features, fewer bugs | `bug-report-triage` — reproduce steps + severity from a report · `pr-description` — clear PR summary from a diff · `standup-update` — yesterday/today/blockers from notes |
| Fitness / wellness | more clients, keep them | `workout-planner` — session plan from goal + equipment · `client-checkin` — weekly check-in message + adjustments · `program-explainer` — plain-words explanation of a plan |
| (No clear match) | — | Reason from the goal alone: identify the 3 most repeated WEEKLY jobs implied by it, one skill each. When in doubt, favor: a reply/communication skill, a planning skill, and a drafting skill in their domain. |
