# Claude Code Easy Install — ติดตั้ง Claude Code แบบง่ายสุด ๆ

[English](README.md) · [Tiếng Việt](README.vi.md) · **ไทย** · [한국어](README.ko.md) · [中文](README.zh.md)

**คำสั่งเดียว → ติดตั้ง Claude Code เสร็จ → ได้ผู้ช่วย AI ส่วนตัวของคุณเอง ในภาษาของคุณ**

> ภาษาอังกฤษและเวียดนามคือภาษาหลักที่รองรับ ส่วนภาษาไทย เกาหลี และจีน รองรับเต็มรูปแบบทั้งใน `/onboard` และเอกสารนี้ ภาษาอื่น ๆ ใช้ได้ผ่านตัวเลือก "อื่น ๆ" ในช่วงสัมภาษณ์

> 📸 *ภาพตัวอย่างจะตามมาเร็ว ๆ นี้*

---

## นี่คืออะไร?

ชุดเริ่มต้น Claude Code ทั่วไปมักโยนไฟล์มาให้คุณแก้เอง แต่ชุดนี้ **สัมภาษณ์คุณ** แล้วเขียนการตั้งค่าให้เอง:

1. ติดตั้ง [Claude Code](https://code.claude.com) ด้วย **ตัวติดตั้งอย่างเป็นทางการของ Anthropic** (เราไม่นำซอฟต์แวร์ของเขามาแจกจ่ายเอง)
2. ติดตั้งปลั๊กอิน `nick-starter` จาก repo นี้
3. คุณพิมพ์ `/onboard` → บทสัมภาษณ์เป็นกันเองจะถามชื่อคุณ อาชีพของคุณ **เป้าหมาย** ของคุณ **ชื่อ** และ **บุคลิก** ของผู้ช่วย → จากนั้นระบบจะเขียนโปรไฟล์เฉพาะตัวและสร้างหน่วยความจำระยะยาวให้
4. จากนั้น — โดยอัตโนมัติทั้งหมด — **ผู้ช่วยจะสร้างสกิลของตัวเอง**: จากอาชีพและเป้าหมายของคุณ มันจะสร้างความสามารถเฉพาะ 3–5 อย่าง (เช่น ช่างภาพจะได้สกิลตอบลูกค้า ทำใบเสนอราคา วางแผนถ่ายงาน) และติดตั้งให้เสร็จก่อนทักทายคุณ
5. มันยังตั้ง **"สมองที่สอง"** ให้ด้วย: คลังโน้ตพร้อมใช้ที่ `~/second-brain` เปิดดูได้ด้วยแอปฟรี [Obsidian](https://obsidian.md) (ตัวติดตั้งจะพยายามติดตั้งให้เองด้วย)

คุณจะไม่ถูกถามคำถามทางเทคนิคเลย — บทสัมภาษณ์ถามแค่เรื่องของคุณ ส่วนเสริมอัตโนมัติทุกอย่างเป็นตัวเลือก: ถ้าติดตั้งไม่ได้จะถูกข้ามพร้อมข้อความสั้น ๆ ที่เป็นมิตร และทุกอย่างที่เหลือยังทำงานปกติ

ไม่ต้องมี Git ไม่ต้องใช้สิทธิ์ admin (Node.js เป็นเพียงส่วนเสริมทางเลือกที่ตัวติดตั้งจัดการเอง)

## ติดตั้ง — Windows

เปิด **PowerShell** (กด Start พิมพ์ "PowerShell" แล้ว Enter) จากนั้นวาง:

```powershell
irm https://raw.githubusercontent.com/supakitkitsathaporn97-collab/claude-easy-install/main/install/go.ps1 | iex
```

## ติดตั้ง — macOS / Linux

เปิด **Terminal** แล้ววาง:

```bash
curl -fsSL https://raw.githubusercontent.com/supakitkitsathaporn97-collab/claude-easy-install/main/install/go.sh | bash
```

## จากนั้น

1. พิมพ์ `claude` แล้วกด Enter
2. ล็อกอินเมื่อเบราว์เซอร์เปิดขึ้น — **ต้องมีแพ็กเกจ Claude แบบเสียเงิน (Pro หรือ Max)**
3. พิมพ์ `/onboard` เลือกภาษา แล้วพบกับผู้ช่วยคนใหม่ของคุณ

## ความต้องการของระบบ

- Windows 10 1809+ / macOS 13+ / Ubuntu 20.04+
- การเชื่อมต่ออินเทอร์เน็ต
- บัญชี Claude แบบเสียเงิน (Pro, Max หรือ Team) — [claude.ai](https://claude.ai)
- **ยังไม่มี Claude?** เริ่มด้วยทดลอง Pro ฟรี 7 วัน → [claude.ai/referral/QbA1I722cA](https://claude.ai/referral/QbA1I722cA) *(ลิงก์แนะนำ — สนับสนุนโปรเจกต์นี้)*

## ในปลั๊กอินมีอะไรบ้าง

| สกิล | ทำอะไร |
|---|---|
| `/onboard` | บทสัมภาษณ์ที่สร้างผู้ช่วยเฉพาะตัว + หน่วยความจำ + สกิลเฉพาะตัว + สมองที่สอง — เป็นภาษาอังกฤษ เวียดนาม ไทย เกาหลี จีน หรือภาษาของคุณเอง |
| `forge-skills` | **สร้างสกิลเฉพาะ 3–5 อย่างสำหรับอาชีพและเป้าหมายของคุณโดยอัตโนมัติ** — รันอัตโนมัติตอนจบ `/onboard`; รันใหม่ได้ทุกเมื่อด้วย `/forge-skills` หรือพูดว่า "เป้าหมายฉันเปลี่ยนแล้ว" |
| `create-skill` | สอนความสามารถใหม่ให้ผู้ช่วยด้วยการอธิบาย — มันจะเขียนและติดตั้งสกิลให้เอง (มาตรฐานคุณภาพเดียวกับ forge) |
| `remember` | บันทึกข้อมูล/ความชอบลงหน่วยความจำระยะยาวของผู้ช่วย |
| `recall` | ค้นหาสิ่งที่คุณเคยบอกไว้ |
| `learn-from-mistakes` | เปลี่ยนคำแก้ไขของคุณให้เป็นกฎถาวร |
| `daily-note` | บันทึกประจำวันแบบง่าย ๆ |
| `work-smart` | ให้ผู้ช่วยวางแผนก่อนลงมือ ไม่ทำขั้นตอนเปลืองเปล่า |
| `personal` | พื้นฐานผู้ช่วยส่วนตัว: บุคลิกคงเส้นคงวา นิสัยจดจำ ความซื่อตรง ขอบเขตความปลอดภัย |
| `leader` | สำหรับงานใหญ่: วางแผน แตกงาน มอบหมาย ตรวจสอบ รายงานคำตอบเดียวที่ชัดเจน |
| `/setup-vault` | สร้าง (ใหม่) คลังโน้ต "สมองที่สอง" ที่ `~/second-brain` — พร้อมใช้กับ Obsidian |
| `obsidian-markdown` · `obsidian-cli` | เขียนและจัดระเบียบโน้ตอย่างถูกวิธี (wikilink แท็ก จัดการไฟล์อย่างปลอดภัย) |

## สมองที่สองของคุณ

`/onboard` สร้างคลังโน้ตที่ `~/second-brain` — ไฟล์ markdown ธรรมดาที่ผู้ช่วยอ่านและเขียนได้ (บันทึกประจำวัน โปรเจกต์ ผู้คน ไอเดีย) เปิดโฟลเดอร์นี้ในแอปฟรี [Obsidian](https://obsidian.md) ("Open folder as vault") เพื่อดูแบบภาพ — ตัวติดตั้งจะพยายามติดตั้ง Obsidian ให้ ถ้าไม่ได้ทุกอย่างก็ยังทำงานเป็นไฟล์ธรรมดา ตัวติดตั้งยังลองอัปเกรด "หน่วยความจำอัจฉริยะ" เสริม (ปลั๊กอินโอเพนซอร์ส [agentmemory](https://github.com/rohitg00/agentmemory)) ถ้าติดตั้งไม่ได้ ผู้ช่วยก็ยังจดจำทุกอย่างผ่านไฟล์ตามปกติ

## คำถามที่พบบ่อย

**นี่เป็นซอฟต์แวร์ทางการของ Anthropic ไหม?**
ไม่ใช่ นี่คือชุดเริ่มต้นอิสระ มันติดตั้ง Claude Code โดยเรียกตัวติดตั้งอย่างเป็นทางการของ Anthropic เอง แล้วเพิ่มปลั๊กอินทับลงไป Claude Code เป็นของ Anthropic

**สคริปต์ติดตั้งปลอดภัยไหม?**
ปลอดภัย — และคุณไม่ต้องเชื่อเราลอย ๆ: อ่าน [`install/go.ps1`](install/go.ps1) และ [`install/go.sh`](install/go.sh) ด้วยตัวเองได้เลย มันทำแค่ (1) เรียกตัวติดตั้งทางการของ Anthropic (2) ลงทะเบียน repo นี้เป็นแหล่งปลั๊กอิน (3) ติดตั้งปลั๊กอิน Windows SmartScreen หรือโปรแกรมแอนติไวรัสอาจเตือนกับสคริปต์ใด ๆ ที่โหลดจากอินเทอร์เน็ต — ซึ่งเป็นเรื่องปกติของวิธีติดตั้งแบบนี้

**เผลอรันซ้ำสองรอบทำไงดี?**
ไม่เป็นไรเลย — สคริปต์รันซ้ำได้อย่างปลอดภัย ส่วนไหนติดตั้งแล้วจะข้ามไปเอง

**เปลี่ยนผู้ช่วยทีหลังได้ไหม?**
ได้ รัน `/onboard` ใหม่ได้ทุกเมื่อ โปรไฟล์เก่าจะถูกสำรองไว้ก่อนเสมอ ไม่มีวันถูกลบ

**มีการเก็บข้อมูลของฉันไหม?**
ไม่มี ทุกอย่างที่บทสัมภาษณ์เขียนออกมาอยู่ใน `~/.claude/` บนเครื่องของคุณเองทั้งหมด
